How to:
1. Open the terminal, come to the directory of the java files
2. input:
   javac *.java
3. input:
   java VMTranslator file|dir
4. Then you will find translated files in the same directory

example:
yangyantekiMacBook-Air:Translator yangyan$ javac *.java
yangyantekiMacBook-Air:Translator yangyan$ java VMTranslator FunctionCalls/FibonacciElement

* All the errors will be printed by the translator
