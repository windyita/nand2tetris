How to:
1. Open the terminal, come to the directory of the java files
2. input:
   javac *.java
3. input:
   java JackCompiler filename|dir
4. Then you will find a new vm file in the same directory

example:
yangyantekiMacBook-Air:CompilerII yangyan$ pwd
/Users/yangyan/Desktop/CompilerII
yangyantekiMacBook-Air:CompilerII yangyan$ javac *.java
yangyantekiMacBook-Air:CompilerII yangyan$ java JackCompiler ArrayTest

* All the errors will be printed by the compiler
